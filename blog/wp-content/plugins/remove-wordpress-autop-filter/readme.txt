== Description ==

This plugin will remove the &lt;p&gt; that is automatically inserted by WordPress the posts.

== Installation ==

1. Download then upload the plugin to wp-content/plugins/
2. Activate the plugin
3. Enjoy a &lt;p&gt;-free blog :)

== Frequently Asked Questions ==

= Why do I need this plugin? =

This plugin is good for you if you need to have your own formatting.  WordPress by default inserts &lt;p&gt; and &lt;/p&gt; for posts and breaks, but maybe you need to do your own formatting.  This plugin will allow for that.
