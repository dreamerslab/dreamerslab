Remove extraneous header tags from your site's head area. Simple user interface that allows you to select what tags to remove.

== Description ==

**Head cleanup** removes tags from your head area. You can control which tags are removed through a simple admin page. Did I miss something? or you found a bug? Let me know!

[WordPress Head Cleanup Plugin at BinaryM.com](http://binarym.com/2010/wordpress-head-cleanup-plugin/)

== Installation ==

1. Upload plugin files
1. Activate plugin
1. Go to Settings > Header Cleanup
1. Select options to remove
1. Save it
1. Done!
